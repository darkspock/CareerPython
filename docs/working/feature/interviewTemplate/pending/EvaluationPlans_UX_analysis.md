# Evaluation Plans - UX Analysis

## 1. User Personas & Goals

### 1.1 Primary Users

| Persona | Role | Primary Goals |
|---------|------|---------------|
| **HR Manager** | Designs evaluation processes | Create reusable Evaluation Plans for different roles |
| **Recruiter** | Manages job positions | Assign plans to positions, override when needed |
| **Hiring Manager** | Reviews candidates | Understand what interviews candidates will go through |

### 1.2 User Goals by Feature

| Feature | User Goal |
|---------|-----------|
| Evaluation Plans | "I want to create a standard interview kit for Frontend roles that I can reuse" |
| Plan Assignment | "I want to quickly assign the right interview kit when creating a job position" |
| Position Override | "This specific position needs an extra security interview - I need to add it" |
| Override Visibility | "I need to see if this position uses standard interviews or has custom ones" |

## 6. States & Edge Cases

### 6.1 Empty States

| State | Message | Action |
|-------|---------|--------|
| No plans exist | "No evaluation plans yet. Create your first plan to standardize interviews across positions." | [Create Plan] button |
| Plan has no templates | "This stage has no interview templates configured." | [Add Template] button |
| No compatible plans | "No plans compatible with this workflow. Create a new plan or change workflow." | [Create Plan] or [Change Workflow] |

### 6.2 Warning States

| Condition | Warning |
|-----------|---------|
| Plan doesn't cover all stages | âš ï¸ "3 stages have no interview templates configured" |
| Changing workflow with plan assigned | âš ï¸ "Changing workflow will remove the current evaluation plan" |
| Deleting plan used by positions | âš ï¸ "This plan is used by 5 positions. They will fall back to workflow defaults." |
| Override removes all templates | âš ï¸ "No interviews will be triggered for this stage" |

### 6.3 Conflict Resolution

| Conflict | Resolution |
|----------|------------|
| Plan deleted while assigned | Position falls back to workflow stage defaults |
| Template deleted from plan | Template removed from plan config, warning shown |
| Workflow changed on position | Plan assignment cleared, user must re-select |
