# Job Position

The goal of the view is to allow stakeholders review, refine a approve the requirements for a job position before it is published.

## Requirements
In this view we should see:
* Custom Fields
* Common Fields
* Any interaction on this job position, edits, stage moves, etc...
* User can add comments to the job position in any stage or globaly.
* User can mark comments as pending or reviewed.
* Comments from other stage should not appear directly. We should only see global and current comments.
